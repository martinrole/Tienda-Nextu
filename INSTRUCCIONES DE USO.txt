INSTRUCCIONES FUNCIONAMIENTO:

CARPETA BACKEND           : Trabaja en localhost 4000

1 - Abrir carpeta BACKEND en editor de código         - (Visual Studio Code)
2 - Abrir una consola y escribir: mongod              - (Debe tener instalado Mongodb en PC)
3 - Abrir otra consola y escribir: mongo              - Para iniciar el servidor
4 - Abrir otra consola y escribir: npm run ejecutar   - Para iniciar el BACKEND



CARPETA FRONTEND ANGULAR : Trabaja en localhost 4200

5 - Abrir carpeta FRONTEND ANGULAR en editor de código   - (Visual Studio Code)
6 - Abrir una consola y escribir: ng serve               - Para iniciar ANGULAR



CARPETA FRONTEND REACTJS : Trabaja en localhost 3000

7 - Abrir carpeta FRONTEND REACTJS en editor de código    - (Visual Studio Code)
8 - Abrir una consola y escribir: npm start		  - Para iniciar REACTJS


------------------------------
EN NAVEGADOR:

9 - Abrir navegador y escribir dirección: http://localhost:4200/     - Muestra la tienda en ANGULAR
9 - Abrir navegador y escribir dirección: http://localhost:3000/     - Muestra la tienda en REACTJS


------------------------
FUNCIONAMIENTO:

10 - Se debe crear un nuevo usuario: Dar click en Registrate! En cualquier proyecto (ANGULAR o REACTJS)
11 - Digitar correo y contraseña creado en el paso anterior para ingresar a la tienda
